## 2. 환경 구성 체크리스트

- [ ] VMware Workstation Player 설치 완료
- [ ] `cloud-lab-vm.zip` 압축 해제 완료 (`cloud-lab.vmx`, `cloud-lab.vmdk` 확인)
- [ ] VM 실행 및 `student` / `a1234` 로그인 성공
- [ ] `docker --version` 실행 → 버전 정상 출력 — 캡처 첨부
- [ ] `kubectl version --client` 실행 → 버전 정상 출력 — 캡처 첨부
- [ ] `minikube start` 실행 → 클러스터 기동 완료
- [ ] `minikube status` 실행 → host/kubelet/apiserver 모두 Running 확인 — 캡처 첨부
- [ ] VM 종료 시 `sudo shutdown -h now` 또는 Player → Power → Shut Down Guest로 정상 종료 (Power Off 강제 종료 금지)
