## [슬라이드 05] Freestyle Job의 한계

"11주차에 만든 `hello-jenkins`, 기억하시죠? 화면에서 클릭으로 Build Steps를 설정했었습니다.

**(문제 제기)** 그런데 이 방식에는 한계가 있습니다. 화면 설정은 다른 사람과 공유하기 어렵고, 빌드
단계가 늘어날수록(체크아웃 → 빌드 → 푸시 → 배포처럼) 화면 설정만으로는 점점 복잡해집니다.

**(해결책 제시)** 이 문제를 해결하는 게 **Jenkinsfile**입니다. Jenkins의 빌드 파이프라인을 코드,
즉 텍스트 파일로 선언하는 방식이에요. 코드로 관리하면 9~10주차에 배운 것처럼 **Git으로 버전
관리**하고, 리뷰하고, 재사용할 수 있습니다."

## [슬라이드 07] Jenkinsfile의 기본 뼈대

"Jenkinsfile을 작성하는 가장 표준적인 방식이 **Declarative Pipeline**입니다. 기본 뼈대는 이렇습니다.

```groovy
pipeline {
  agent any
  stages {
    stage("Build") {
      steps { ... }
    }
  }
}
```

`pipeline { }`이 전체를 감싸는 최상위 블록입니다. `agent any`는 이 파이프라인을 어떤
서버(에이전트)에서 실행할지 지정하는데, `any`는 '아무 에이전트나 써도 된다'는 뜻입니다. `stages`
안에 단계별 작업을 순서대로 나열합니다."

---

## [슬라이드 08] Stage로 단계 나누기

"파이프라인을 하나로 뭉치지 않고 여러 **stage**로 나누는 이유가 있습니다.

예를 들어 `Checkout`(코드 받기) → `Build`(빌드) → `Push`(업로드) → `Deploy`(배포)처럼 나누면, 각
단계가 무슨 일을 하는지 명확해집니다.

**(실용적 이점)** 그리고 Jenkins UI에서 각 stage가 성공했는지 실패했는지 **시각적으로** 확인할 수
있습니다. 나중에 파이프라인이 실패하면, 어느 stage에서 문제가 생겼는지 한눈에 알 수 있는 거죠."

---

## [슬라이드 09] 환경변수와 Credential

"마지막으로 환경변수와 Credential입니다.

`environment { }` 블록으로 파이프라인 전체에서 쓸 환경변수를 선언할 수 있습니다. 그리고
`$BUILD_NUMBER`처럼 Jenkins가 기본으로 제공하는 변수도 자동으로 쓸 수 있어요. 11주차 마지막
실습에서 이미 한 번 써봤죠.

**(보안 개념 강조)** `withCredentials { }`는 Docker Hub 비밀번호 같은 민감한 정보를 코드에
**그대로 노출하지 않고** 안전하게 사용하는 방법입니다. 비밀번호를 Jenkinsfile에 평문으로 적으면
GitHub에 그대로 올라가 버리니, 반드시 이 방식으로 다뤄야 합니다."

## 강의자 참고 — 예상 질문 & 답변

| 예상 질문 | 답변 가이드 |
|---|---|
| "Declarative Pipeline 말고 다른 방식도 있나요?" | Scripted Pipeline이라는 더 자유로운 문법도 있습니다. 하지만 Declarative가 더 읽기 쉽고 표준적이라 실무에서도 더 널리 쓰이며, 이 과목에서는 Declarative만 다룹니다. |
| "agent any 말고 다른 옵션도 있나요?" | 특정 서버나 특정 라벨이 붙은 에이전트를 지정할 수도 있습니다. 이 과목에서는 별도 에이전트를 구성하지 않으므로 항상 `any`를 씁니다. |
| "Jenkinsfile은 어디에 저장하나요?" | 보통 프로젝트의 GitHub 저장소 안에 `Jenkinsfile`이라는 이름으로 함께 저장합니다. 다음 3차시에 이 방식(저장소 안의 Jenkinsfile을 Jenkins가 직접 읽어오는 방식)을 배웁니다. |
| "Killercoda 링크가 언제 만들어졌는지에 따라 제한 시간이 달라지나요?" | 아니요. 60분 타이머는 링크 생성 시점이 아니라 각 학생이 시나리오를 '시작'한 순간부터 개별적으로 시작됩니다. 링크 자체는 만료되지 않으므로, 비동기로 늦게 접속하거나 여러 학생이 같은 링크를 써도 각자 독립된 환경과 타이머를 받습니다. |
