## 00_공통자료 — 특정 주차에 속하지 않는 자료

| 폴더 | 용도 | 상태 |
|---|---|---|
| `강의계획서/` | 전체 커리큘럼 요약본, 학기 전체를 한눈에 보는 자료 | ✅ 작성됨 |
| `평가_및_공지/` | 평가 방법 안내, 학기 공지, 과제 제출 양식 템플릿 | ✅ 과제 제출 양식 템플릿 작성됨 |
| `오류_자가진단_가이드/` | 주차 공통으로 반복되는 오류 유형과 자가진단 절차 | ⏳ 미작성 |
| `입문자_보충자료/` | 클라우드·리눅스를 처음 접하는 학생을 위한 사전 용어·개념 보충 | ⏳ 미작성 |

## 실습 주차 폴더 구조 (0~7, 9~12주차)

0주차와 1~7·9~12주차는 아래처럼 번호가 매겨진 하위 폴더로 구성됩니다. 폴더명 앞의 숫자가 곧
차시 진행 순서(개념설명 → 실습 → 과제 → 제출)를 반영합니다.

```
NN주차_주제/
├── 01_강의자료/          # 1~3차시 강의용/배포용 pptx (강의용 = 교수자 시연 슬라이드 포함)
├── 02_강의대본/           # 1~3차시 슬라이드별 상세 스크립트(md) — 괄호 지시사항 포함
├── 03_실습가이드_PDF/     # 단계별 스크린샷 포함 따라하기 문서(Killercoda/VM 구분)
├── 04_템플릿코드/         # YAML·HCL·Jenkinsfile 등 실습용 뼈대 코드
├── 05_Killercoda_링크/    # 해당 주차 전용 Killercoda 커스텀 시나리오 URL
├── 06_로컬VM_과제복습/    # VM에서 진행하는 과제/복습 실습 안내
├── 07_오류FAQ/            # 그 주차에서 자주 발생하는 오류 Top 5와 해결법
├── 08_퀴즈/               # 개념 이해 확인용 자동채점 퀴즈(5~10문항)
├── 09_체크리스트/         # 실습 완료 확인용 체크리스트
└── 10_과제제출/           # 학생 제출용 양식 (00_공통자료 템플릿의 주차별 인스턴스)
```

- **0주차**는 위 패턴 대신 `01_강의자료`, `02_강의대본`, `02_VM구성스크립트_Multipass_Vagrant`,
  `03_환경검증가이드`, `04_오류FAQ`, `05_체크리스트`로 구성됩니다(과제 제출이 아닌 환경 구성 확인이
  목적이므로 `10_과제제출` 대신 `05_체크리스트`에 환경구성 확인서를 둡니다).
- **13~14주차**(통합 실습)는 100% VM 전용이라 `05_Killercoda_링크`가 없고, 대신
  `05_로컬VM_통합실습_전용`이 그 자리를 대체하며 이후 번호가 한 칸씩 당겨집니다
  (`06_오류FAQ` · `07_퀴즈` · `08_체크리스트` · `09_과제제출`).

## 파일 명명 규칙

- 강의자료(pptx): `{N}차시_{주제}_강의용.pptx` / `_배포용.pptx` — 강의용은 교수자 시연 배지가 있고,
  배포용은 학생에게 그대로 배포하는 버전(시연 배지 제거)
- 강의대본(md): `{N}차시_강의대본.md` — 슬라이드 번호별 스크립트 + 하단에 "강의자 참고 — 예상 질문 & 답변" 표
- 0주차만 예외적으로 차시 구분 없이 `강의대본.md` 단일 파일
