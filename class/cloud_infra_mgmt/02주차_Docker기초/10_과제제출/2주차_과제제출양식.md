## 2. 실습 체크리스트

### 1차시 — 컨테이너 명령어 심화 (★ 필수)
- [ ] `docker ps -a` 실행 결과 — 캡처 첨부
- [ ] `docker logs {컨테이너명}` 출력 화면 — 캡처 첨부

### 2차시 — Bind Mount (★ 필수)
- [ ] `docker run` 으로 Bind Mount 컨테이너(`webvol`) 실행
- [ ] `curl localhost:8080` 결과 화면 — 캡처 첨부
- [ ] `webvol` 컨테이너를 삭제하지 않고 유지했음

### 3차시 — Named Volume·데이터 영속성 (★ 필수, 2주차 종합)
- [ ] `webvol`의 index.html 수정 후 즉시 반영되는 화면 — 캡처 첨부
- [ ] Named Volume으로 만든 `webvol2` 컨테이너 실행 결과 — 캡처 첨부
