## 2. 실습 체크리스트

### 1차시 — Service 개념 (★ 필수)
- [ ] `myapp` Deployment가 정상 실행 중인지 확인
- [ ] `kubectl get pods -o wide` 실행 결과(Pod IP 확인) — 캡처 첨부

### 2차시 — NodePort 실습 (★ 필수)
- [ ] `kubectl expose deployment myapp --type=NodePort --port=80` 실행
- [ ] `minikube service myapp --url`로 URL 확인
- [ ] curl로 접속한 결과 — 캡처 첨부

### 3차시 — Endpoints·라운드로빈 (★ 필수, 5주차 종합)
- [ ] `kubectl get endpoints myapp` 실행 결과 — 캡처 첨부
- [ ] 반복 curl(`for i in $(seq 1 10); do curl $(minikube service myapp --url); done`) 실행 결과 — 캡처 첨부
