# 클라우드인프라관리 — 주차별 실습 과제 제출 양식 (마스터 템플릿)

이 문서는 매 주차 과제 제출 양식의 공통 뼈대입니다. 각 주차의 `10_과제제출`(13~14주차는
`09_과제제출`) 폴더에는 이 템플릿을 바탕으로 그 주차의 실습 체크리스트를 채운 인스턴스 파일이
들어 있습니다. 새 주차 자료를 만들 때는 이 템플릿을 복사한 뒤 "2. 실습 체크리스트"만 해당 주차
강의대본의 "VM 안내"·"결과 제출" 절 내용에 맞게 교체하면 됩니다.

## 2. 실습 체크리스트

> 아래는 예시 형식입니다. 실제 항목은 각 주차 인스턴스 파일을 확인하세요.

- [ ] 실습 항목 1 — 캡처 첨부: (파일명 또는 이미지 삽입)
- [ ] 실습 항목 2 — 캡처 첨부: (파일명 또는 이미지 삽입)
- [ ] (해당 시) GitHub 저장소 링크: `https://github.com/{아이디}/{저장소명}`

## 3. 자가 점검

- [ ] 위 체크리스트 항목을 모두 VM(또는 지정된 환경)에서 직접 실행하고 캡처했다
- [ ] 캡처 화면에 명령어와 결과가 함께 보이도록 캡처했다 (터미널 프롬프트 포함)
- [ ] 파일명에 학번을 포함했다 (예: `202312345_3주차2차시.png`)

## 4. 특이사항 / 질문 (선택)

> 실습 중 막혔던 부분, 궁금한 점, 참고한 자료 등을 자유롭게 남겨주세요.

---

## 제출 방법 안내 (학생용 공통 안내문)

1. 위 체크리스트의 각 항목에 대해 **명령어와 실행 결과가 함께 보이는 터미널 캡처**를 준비합니다.
2. 이 문서를 복사해 학번을 포함한 파일명(예: `202312345_3주차2차시_과제.md` 또는 `.hwp`/`.docx`)으로
   저장하고, 캡처 이미지를 첨부하거나 삽입합니다.
3. LMS 과제 게시판의 해당 주차 게시물에 제출합니다.
4. 제출 기한은 다음 차시 수업 시작 전까지이며, 각 차시 과제는 독립적으로 제출할 수 있습니다
   (한 차시를 완료하지 못했다고 다른 차시 제출이 막히지 않습니다).
5. GitHub/Docker Hub/Terraform Cloud 링크를 제출하는 주차는, 저장소·리포지터리가 **Public**으로
   설정되어 있는지 반드시 확인하세요 (Private이면 채점자가 접근할 수 없습니다).

## 평가 반영

주차별 실습 과제는 전체 성적의 30%를 차지합니다 (중간고사 30%, 기말고사 30%, 출석 10%와 함께).
자세한 평가 방법은 [`전체_커리큘럼_한눈에보기.md`](../강의계획서/전체_커리큘럼_한눈에보기.md)의
"4. 평가 방법"을 참조하세요.
